static void WpfHandler.UI.LocalizationHandler.LoadDictionaries ( params CultureInfo []  culturesOrder)
static

Scaning for language dictionaries in XAML files stored by Plugins.Constants.PLUGINS_DIR, and loading them to Merged dictionaries. Loading only relative to the new culture if found. Leave a previous culture if not.

Require files format: *.lang.CULTURE_CODE.xaml, where culture code equal current translation of the app. Example: plugin.feed.lang.en-US.xaml

Parameters
culturesOrderArray of cultures reuested to loading in the priiority oreder.
