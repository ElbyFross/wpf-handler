◆ AnimationType

enum WpfHandler.UI.Controls.SwitchPanel.AnimationType
strong

Type of animation.

Enumerator
None 

Animation not defined. Control will switch to the next element immediately.

AlphaSwipe 

Switching between elements via making current element transparency and rolling of the next element from the border of the current control.

DarkSwipe 

Shadowing the background and swipe current element out of the control's border. Removing the shadow when timeline is finished.
